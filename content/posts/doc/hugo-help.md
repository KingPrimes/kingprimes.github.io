# Hugo Help

### 创建文档
``` bash
hugo new content [path]/[name].md
```

### 启动本地Hugo服务器
``` bash
-- 以下命令启动包含草稿内容
hugo server -D
hugo server --buildDrafts
hugo server -D --disableFastRender
```
