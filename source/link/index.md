## 友链申请

欢迎交换友链！请在下方评论区按照以下格式提交你的站点信息：

```yaml
- class_name: 分类名称（例如：技术博客）
  class_desc: 分类描述（例如：热爱编程的开发者）
  link_list:
    - name: 站点名称
      link: https://你的网站地址/
      avatar: https://你的头像地址/
      desc: 站点描述
```
